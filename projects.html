Time and Space Efficient Construction of CCDBG

Working on an Advanced Project involving reducing the space and time complexity of Colored Compacted De Bruijn Graph. The projecct is being done under the guidance of Prof. Rob Patro. We are using the tool TwoPaCo. The output of this tool is given to Pufferfish and we are researching ways to reduce the time and space constraints to make the step more efficient for large scale genomes.
Used: C++

The Searching Pacman

Its a famous search problem proposed by University of Berkeley as an introductory project for Artificial Intelligence course. Did this project as part of CSE537.

Healthcare Financial Website

Worked in a team of 4-5 members to develop an interactive website which helps people find the best healthcare plan as per their demographics and personal information. The project involved scrum calls with the client, agile methodology for the tasks, rigorous testing and weekly deployment. The front-end communicated with the middleware using REST API calls and the data was fetched by the middleware from the back-end and provided to the fron-end for display purposes. We also developed iOS and Android apps for the same Web-application.

Onine Marketplace Analysis

Worked on creating an online marketplace analytics website, in a team of 3. We procured shipping, sales and geographical data from sample marketplaces built on Magento and Shopify, and performed analytics over the data to recommend which products should be sold at which locations, which locations are experiencing low sales, which products are selling well in the market, etc. Web Hooks were used to periodically read data from the marketplaces and RabbitMQ was used to queue the incoming data for a large data set.
Used: PHP, Python, AngularJS, MySQL, RabbitMQ, Highcharts, Cron-jobs

Marathi to English Translator for Simple Sentences

Developed a Rule-Based Language translator to translate simple sentences from Marathi to English Language. Effectively generates semantically correct English sentences. Achieved an accuracy of 89%.
Worked in a group of 4 on this project, using Java, JSP and HTML. We used a POS tagger maintained by TDIL. Created a rearrangement algorithm which generates a semantically correct English sentence from a given Marathi sentence using Word to Word Translation and Rearrangement of the Words.
Used: Java, JSP, HTML
